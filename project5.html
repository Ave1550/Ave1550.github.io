Turtle Project

About the Project

This project is about creating a graphical scene using Pythonâ€™s Turtle graphics module.I used the provided building block functions (draw_rectangle, draw_square, draw_triangle, draw_circle, draw_polygon, draw_curve, etc.) to create functions that draw composite shapes such as a house, flowers, trees, etc. I refactored it by creating helper functions for different parts of the scene. Instead of writing all the drawing steps inside one big function, I split the tasks into smaller functions like draw_house, draw_sun, and draw_cloud. Each of these functions calls the basic shape-drawing functions, so the code is less repetitive and easier to understand.

Sample Code

This function creates a house using rectangles and other shapes.

def draw_house(t, x, y, width, height): # House body jump_to(t, x, y) draw_rectangle(t, width, height, 'red') # Roof jump_to(t, x - 10, y - 10) draw_triangle(t, width + 20, 'brown') # Door door_width, door_height = 20, 40 jump_to(t, x + width//2 - door_width//2, y - height + door_height) draw_rectangle(t, door_width, door_height, 'yellow')

Final Scene

Here's the completed turtle graphics scene featuring a house, fluffy clouds, a bright sun.
